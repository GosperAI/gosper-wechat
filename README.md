# Gosper WeChat

Standalone WeChat transport bridge for Gosper.

It runs as a resident Node.js service, uses iLink QR login, `getupdates`, and
`sendmessage`, then forwards inbound WeChat messages to Gosper.

See [README.zh_CN.md](./README.zh_CN.md) for quick start and
[docs/usage.zh-CN.md](./docs/usage.zh-CN.md) for the full setup guide.
